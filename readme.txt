== Description ==

Mit dem Wordpress Plugin mmyReferences könnt Ihr schnell und einfach über Tags (<ref>URL</ref>) Quellenangaben in Eure Beträge einfügen. mmyReferences basiert auf dem Wordpress Plugin TicTagTo von Olaf A.Schmitz - http://blogshop.de/18012005,199.

== Installation ==

1. Datei mmyref.php in das Wordpress-Verzeichnis wp-content/plugins einfügen.
2. Folgenden Eintrag in Eure CSS-Datei einfügen und, wenn Ihr möchtet, anpassen:
.mmyref {
font-family: arial, verdana, tahoma, helvetica, sans-serif;
font-size:10px;
padding:0px 0px 0px 20px;
margin:0px 0px 0px 230px;
float:right;
}
3. In den Administrationsbereich von Wordpress einloggen und unter dem Menüpunkt Plugins das Plugin mmyReferences aktivieren.
